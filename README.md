# Arave - Text Editor (work in progress)
## Installation
### Build from scripts
0\. Check these requirements
+ Node.js ^18.17.0
+ npm with npx

1\. Clone this repo
```
$ git clone https://github.com/ihasq/arave.git
$ cd arave
```

2\. Run build & install script
```
$ node scripts/build
$ node scripts/install
```

3\. Run Arave at anywhere
```
$ arave
```
